### For Gitbook

- `###` for subheadings
- Include filenames and Component names in backticks
- Keep EFA Logo on Front Page
- Directions exceeding 2 steps should be numbered/bulletedâ€¦.
- Make sure all code is in a code block (three backticks and the language - see markdown docs) 
- Resources/Works Cited at end

### For Contributing to this particular Gitbook
- Commit messages should start with the letter for the badge, followed by the week number, followed by the day number or file name
- Example commit messages: 
    - G.1.1: Modifed morning challenge
    - B.2.Home: Updated week 2 guide
    - G.2.Challenges: Added a challenge for queues
